For each of the following SQL queries, submit your SQL and the resulting table produced by the query.  I strongly suggest that you try out your queries using a database engine such as H2.  See the tutorial on using the H2 engine for more information.  As a hint, all of these exercises will start as follows, so that your job on this exercise is to replace the ... in the WHERE clause with the correct (logic-inspired) expression:

  2. [5 points each part] Let p be the proposition "John is healthy", q be the proposition "John is wealthy", and r be the proposition "John is wise".  Write the following English statements in symbolic form using negation, and, or.  Your answer should be some combination of p, q, r, the negation symbol, and, or.  (See exercise 8 on page 15 of the Epp textbook for similar questions, some of which are solved).

    2. John is neither healthy, wealthy, nor wise.
