Assignment

Answer the following questions:

  1. [5 points] What specific result would the following SQL query produce?  You should list the entries that would appear in the table of results if this query were run on a database engine using the university.sql data.  How would you describe the entries in the table produced by the query?  Your answer should describe the attributes listed in the query as well as the characteristic(s) that the entries share in order to have been selected by the query.

  1. [4 points] Let A = {1, 2, 3} and B = {Djengo, Simone, Hugin}.  Compute the Cartesian product A x B.
  2. [4 points] What is the power set of {a, {a}}?
  3. [5 points] Compute P({Djengo, Simone, Hugin}) - P({Djengo, Simone, Joon}).  Hint: Can you find an easier way to do this than just computing the two power sets independently and taking the difference?
  4. [9 points each] For each of the following relations R(x,y) state whether it is reflexive, symmetric, and/or transitive.  For each of the three properties briefly argue that the relation has the property or give a counterexample showing it does not.
    1. "x and y were born on the same date" over the universe of all humans
    2. "x is an ancestor of y" over the universe of all humans
    3. "x = y+1" over the universe of the integers {..., -3, -2, -1, 0, 1, 2, 3, ...}
    4. "x * y >= 0" over the universe of real numbers
  5. Consider the relation R(x,y) = "|x| = |y|" over the universe of real numbers where |x| is the absolute value of x.  For example, |-3.1415| = 3.1415 and |8.75| = 8.75.
    1. [9 points] Show that R is an equivalence relation.
    2. [3 points] What are the equivalence classes of R?

[10 points each] For each of the following relations R(x,y) state whether it is reflexive, anti-symmetric, and/or transitive on the real numbers.  For each of the three properties briefly argue that the relation has the property or give a counterexample showing it does not.  Identify those that are ordering relations (e.g. have all three properties).
    1. R(x,y) = "x + y = 0"
    2. R(x,y) = "x * y >= 0"
    3. R(x,y) = "x = 2y"
